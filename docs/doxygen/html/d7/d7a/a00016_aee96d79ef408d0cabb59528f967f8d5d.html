Configuración.

Si se nos da un valor para una variable, siempre la colocamos en el archivo de configuración del proyecto.

El archivo llevara la variable $config[<variable>][<valor>];

Para acelerar el proceso guardamos en cache

Parámetros:
$nombre_variableNombre de la variable
$valorValor para la variable
Devuelve:
En caso de pedir valor de variable: False en caso de no encontrar o su valor En caso de darnos un valor TRUE o FALSE dependiendo del exito de la operación
