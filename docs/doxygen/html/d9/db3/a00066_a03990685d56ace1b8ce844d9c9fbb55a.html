coincide_archivo_css ( $  archivo) [inherited]

Buscamos si los archivos css de condiciones, entran dentro de la plataforma, el navegador y versión actual.

El mecanismo se basa en archivos css con nomenclatura:

[plataforma]_[navegador]_[version].css

Puede añadirse un all para especificar todas, ejemplo:

  • Linux_all_all.css Para todos los que vengan de linux.
    • all_Safari_all.css Navegadores safari
    • all_Firefox_3.css Firefox version 3
Tareas pendientes:
Implementar comparación de versión
Parámetros:
$nombre_archivoNombre del archivo
$plataformaPlataforma en la que estamos
$navegadorNavegador actual
$versionVersión del navegador
Devuelve:
TRUE/FALSE
