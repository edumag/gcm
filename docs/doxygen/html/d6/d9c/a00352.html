Sistema de plantilla.

Añadir contenido sobre una plantilla html

ejemplo de uso:: 

$plantilla = new Plantilla(GCM_DIR.'modulos/gcm/temas/html/principal.html', $this);
echo $plantilla->procesaPlantilla();

En la clase llamadora tendra que haber un metodo llamado procesaContenido($elemto) que retorne el contenido correspondiente.

EL archivo de plantilla html debe tener entre corchetes el nombre del elemento, {elemento} IMPORTANTE: Solo debe tener caracteres alfanumericos.

En caso de tener un elemento llamado contenido, se procesara el último, esto nos permite que cuando procesemos contenido haya pasado ya por todo lo demas y si hay algún mensaje que contenido deba de presentar lo tendra en cuenta.
