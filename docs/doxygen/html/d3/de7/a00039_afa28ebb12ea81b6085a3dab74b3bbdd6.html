seleccion_idioma ( )

Seleccionamos idioma con el que se presenta la página.

Este metodo detecta idioma actual y genera el array de los literales

El idioma puede venir por GET o POST ['idioma'] o puede estar en la variable de sessión $_SESSION[$this->proyecto."-idioma"]

Nota:
Es importante que se ejecute este metodo al inicio, para comenzar sabiendo antes de nada el idioma que se desea.
