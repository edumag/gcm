procesaPlantilla ( )

Procesamos la plantilla.

Recogemos el contenido de cada elemento para despues remplazar los elementos de la plantilla con el contenido.

Primero elementos vitales primeros despúes elementos vitales últimos y por último los elementos que encontrmos.

Hacemos que contenido se reemplace el último para no dar problemas al editar el archivo de plantilla con los elementos dentro de ella.
