El PageRank de su sitio web es muy bajo. 
PageRank™
 (también llamado PR) es un algoritmo de análisis de enlaces usado por 
Google™ para evaluar la popularidad/autoridad de un sitio web. El 
PageRank va de 0 a 10. Un sitio web nuevo empezaría con PR0 y un sitio
