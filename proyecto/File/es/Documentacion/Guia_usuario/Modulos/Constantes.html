El módulo de constantes nos permite tener definidos unos valores sobre unas variables que podremos añadir a los contenidos.

Un ejemplo practico son por ejemplo los precios, podemos tener definidos unos precios e incluir la constante en cualquier contenido, como por ejemplo una lista de precios y en varios idiomas, cambiar el precio en cada pagina donde lo hayamos puesto implicaría editar una por una, pero con este sistema vamos directamente a modificar la constante y automáticamente se mostrara en cada lugar el nuevo valor.
