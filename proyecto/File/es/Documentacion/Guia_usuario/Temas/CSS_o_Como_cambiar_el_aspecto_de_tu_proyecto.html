proyectos.css

El archivo proyecto.css es generado automaticamente desde gcm con los archivos css de cada módulo o en caso de haberlo con el fichero correspondiente en el tema utilizado.

Añadir imagen desde un archivo css:

farbtastic .overlay {
background: url(<?=Router::$dir.$this->ruta('temas','img','mask.png')?>) no-repeat;
}

Añadir icono desde archivo css:.

.admin ul {
list-style-image: url(<?=Router::$dir.$this->ruta('temas','iconos','16/alerta.gif')?>)
}

Iconos

En el caso de iconos, definimos el nombre del icono sin extensión

background: #F5F3BA url(<?=Router::$dir.$this->icono('alerta')?>) no-repeat 10px 10px ;

Colores

Los colores son guardados en el archivo gcm/modulos/temas/css/colores.php y pueden ser incluidos en los css personalizados con el metodo:

<?=$this->color('nombre del color')?>

Temas

Se puede modificar y crear nuevos temas y modificar las plantillas html desde el menú: Administración->Editor de temas
