Desde la url

Podemos cambiar de tema simplemente añadiendo ?tema=admin por ejemplo a la url, esto nos guardara el tema que queremos en la sesión actual.

Si queremos volver al tema por defecto añadimos ?tema=

Dede los módulos

Para muchos diseños es necesario tener un tema para administrar el proyecto en estos casos debemos tener un tema denominado admin que sera buscado por muchos modulos para poder presentar la información de forma más adecuada.

global $gcm;
$gcm->tema = 'admin';

Activa el tema admin
