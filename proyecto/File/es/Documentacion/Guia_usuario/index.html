Si estas administrando un proyecto con GCM hay algunas características a tener en cuenta.

Registrase en el proyecto

Coloca la url de tu proyecto en tu navegador más registro, ejemplo: http://miproyecto/registro se te presentara el formulario para poder entrar a administrar.

Gestionar contenido

Una vez en tu proyecto podrás ver un menú arriba del todo para comenzar a añadir, modificar o borra contenido.

proyecto

La primera opción del menú contiene el nombre del proyecto y te permite cambiar entre el tema de administración o el tema por defecto. Permitiendo facilitar la administración ya que los temas por defecto puede que carezcan de secciones con contenido imprescindible para administrar cómodamente.

Contenido

Desde aquí podemos gestionar el contenido.

Administración

Dependiendo de los módulos que tengamos activados se nos mostrara como administrar imágenes, comentarios, literales, constantes, etc..
