Cuando estemos editando un contenido existente o creando nuevo contenido se nos presentara un cuadro con un editor web que nos facilitará añadir contenido en formato html sin tener que ser conocedores de este lenguaje.

Menú Contenidos

  • Nuevo:  Creación de nuevo contenido
  • Editar: Para editar el contenido en el que nos encontramos.
  • Traducir: Para traducir el contenido en el que nos encontramos
  • Borrar: Borar contenido actual
  • Renombrar / Mover: Cambiar de nombre o mover a otra sección
  • Guardar como: Guardar una copia en otro lugar
  • Nueva sección: Crear nueva sección
  • Renombrar sección: Cambiar nombre o mover sección
  • Borrar sección: Borrar sección completa (Incluye contenido)
  • Ver todo: Una lista rapida de todos nuestro contenido
