Gcm tiene su forma particular de tratar los idiomas, buscando la facilidad para el usuario el sistema permite activar tantos idiomas como se desee.

Mientras el usuario no seleccione otro idioma se le presentara la pagina en el idioma que tenga configurado en el navegador, en caso de no tener este idioma dentro de los idiomas activados se mostrara en el idioma predeterminado.

Todos los idiomas activados serán seleccionables por el usuario para realizar el cambio.

Si el contenido demandado aun no tiene una traducción se le mostrara al usuario el contenido en el idioma predeterminado, con un mensaje que le indica que aun no hay una traducción.
