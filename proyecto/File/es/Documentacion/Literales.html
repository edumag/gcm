Los literales nos sirven para tener una traducción en cada idioma de palabras que se repiten en cada uno, un ejemplo sería `sección` o `titulo`, si convertimos estas palabras en literales, podremos definir su traducción en los diferentes idiomas que tengamos. Y nosotros añadiríamos {L{sección}} Lo cual provocaría que se buscara la traducción en el idioma que el usuario tiene especificado.

Su mayor utilidad la encontramos con los nombres de los documentos, ya que es igual para cada idioma ya que depende del gestor de archivos, lo que nosotros hacemos es crear literales para cada documento. Esto se lo hace gcm de forma automática así que no tengamos que estar pendiente.

Por ejemplo al crear un nuevo documento gcm nos pide el nombre del mismo, no obstante el guardara el documento con el nombre modificado, cambiando espacios en blanco por guiones bajos y quitando acentos o elementos que puedan ser extraños para el servidor, pero al mismo tiempo crea un literal y se cuando se lista en contenido o se añade el título se presenta el literal.

Para administrar los literales tenemos un panel que nos facilita la modificación.
