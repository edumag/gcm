Mediante Eventos podemos interactuar entre los módulos.

Un ejemplo de como desde un módulo enviamos un email con el módulo enviomail

$args['cuerpo']  = "\nUsuario: ".$resultado['usuario'];
$args['cuerpo'] .= "\nMail: ".$resultado['mail'];
$args['cuerpo'] .= "\nTexto: ";
$args['cuerpo'] .= "\n".$resultado['texto'];
$args['asunto'] = 'Comentario añadido';

$gcm->event->lanzar_accion_modulo('enviomail','enviar_email','comentario_añadido',$args);
