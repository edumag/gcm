Mediante Eventos podemos interactuar entre los módulos.

Un ejemplo de como desde un módulo enviamos un email con el módulo enviomail

$args['cuerpo']  = "nUsuario: ".$resultado['usuario'];
$args['cuerpo'] .= "nMail: ".$resultado['mail'];
$args['cuerpo'] .= "nTexto: ";
$args['cuerpo'] .= "n".$resultado['texto'];
$args['asunto'] = 'Comentario añadido';

$gcm->event->lanzar_accion_modulo('enviomail','enviar_email','comentario_añadido',$args);

Utilizar la instancia de un módulo

$pdo = $gcm->event->instancias['trabajos']->getPDO();

Todos los módulos que hayan sido inicializados estaran disponibles desde Gcm.

Recogida de parametros

Como los parametros pueden venir en diferentes formatos, como un array o como una cadena en forma de url, la función recoger_parametros() dentro de helpers.php nos devolvera venga como venga en una array con el formato $args[variable] = valor

$args = recoger_parametros($args);
