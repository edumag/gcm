Para la creación de test en los módulos que vayamos creando, tenemos un metodo heredado de Modulos que nos facilita la tarea.

   /**
    * Comprobar funcionamiento
    */

   function test() {

      $this->ejecuta_test('Conexión con base de datos', $this->pdo->getAttribute(constant("PDO::ATTR_DRIVER_NAME")), 'mysql' );

      }

Si creamos un metodo test en el módulo, automaticamente sera llamado desde el evento test que podemos llamar desde el menú de administración.

El metodo ejecuta_test solo comprueba que el segundo parametro y el tercero coincidan sino es así nos mostrara mensaje de error.
