Desde el módulo mapas necesitamos que uno de los campos de configuración del módulo tuviera el editor web activado.

Desde MapasAdmin.php pedimos a gcm que nos incluya el javascript del módulo y que añada la librería externa de tiny_mce

   function configuracion($e, $args=FALSE) {

      global $gcm;

      $this->javascripts('editorweb.js');
      $this->add_ext_lib('js',Router::$base.GCM_DIR.'lib/ext/tiny_mce/tiny_mce.js');

      // Añadimos panel con iconos seleccionables
      $gcm->event->accion2evento('columna','mapas','seleccionar_iconos',3);
      parent::configuracion($e, $args);
      }

En el archivo editorweb.js dentro de addLoadEvent para asegurarnos que solo se ejecute al cargar la página, creamos las instancias de tinyMCE para todos los textareas con class GC_contenido

addLoadEvent(function(){
   tinyMCE.init({
      mode : "textareas",
      editor_selector : "GC_contenido",
      theme : "simple",
      languages : 'es',
      });
   });
