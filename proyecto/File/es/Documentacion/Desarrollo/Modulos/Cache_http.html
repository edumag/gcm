Se genera cache para las paginas que nos permite presentarlas más rapidamente.

Es importante que si se genera una modificación sea notificado al módulo cache_http para borrar los archivos relacionados.

Ejemplo al añadir un comentario en el módulo comentarios, insertamos la siguiente linea para que se borre el cache y se presenta la página ctualizada la proxima vez que se presente.

$gcm->event->lanzar_accion_modulo('cache_http','borrar','comentario_insertado',Router::$url);
