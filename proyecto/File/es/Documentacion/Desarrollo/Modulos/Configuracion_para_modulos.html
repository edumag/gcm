Generar archivos de configuración de módulos en la carpeta del mismo:

<modulo>/config

Crear archivo config.php, ejemplo:

<?php
$config['variable']='valor';
$config['array'][]='valor1';
$config['array'][]='valor2';
?>

Crear archivo de descripciones para los idiomas, como minimo el que es por defecto, ejemplo:

Archivo: <módulo>/config/config_es.php

<?php
$config_DESC['variable']='Valor para la variable';
$config_DESC['array']='Listado de valores para el array';
?>

Ahora desde el módulo podremos ir a buscar los valores, ejemplo:

$tema_conf = $this->config('tema_actual');

y el administrador podra configurarlos llamando al modulo desde el navegador: <proyecto>/<módulo>/configuracion
